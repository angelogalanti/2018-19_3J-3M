## HTML
- struttura di un file HTML
```html
<html>
	<head>
		<title>
			titolo
		</title>
	</head>
	<body> <!-- parent di p-->
		<p> <!-- children di body-->
			testo
		</p>
	</body>
</html>
```
- [esempio di utilizzo dell'attributo id in CSS](https://www.w3schools.com/css/tryit.asp?filename=trycss_syntax_id)
